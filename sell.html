매도의 신

EXIT-LADDER-ENGINE · v1.1 · 5-STATE

매도는 고점을 맞히는 일이 아닙니다. 수익을 계좌에 남기는 일입니다. 욕심 때문에 수익을 모두 반납하거나, 두려움 때문에 너무 일찍 전량 매도하지 않도록 내가 정한 가격과 비율로 매도 단계를 관리합니다.

분할 매도 계획이 지금 어느 단계인지 확인합니다. 가격이나 비율을 추천하지 않습니다.

// 처음이신가요
매도 계획은 3가지만 정하면 됩니다.
1
언제 일부를 줄일지 — 수익률이나 직접 가격으로 조건을 정합니다.
2
그때 얼마나 줄일지 — 보유 수량 중 줄일 비율을 정합니다.
3
언제 도달한 것으로 볼지 — 장중에 가격이 닿으면 볼지, 종가로 확인할지 정합니다.
정답으로 정해진 숫자는 없습니다. 조건을 가깝게 두면 수익을 일찍 확정할 수 있지만 이후 상승에 참여할 수량이 줄어듭니다. 조건을 멀리 두면 상승에 더 오래 참여할 수 있지만 이미 발생한 수익을 더 많이 반납할 수 있습니다.
// RULE — 먼저 확인
이 도구는 아래 규칙으로 판정합니다.
  • 1매도 가격과 비율은 사용자가 직접 정합니다.
  • 2한 번에 전량 매도하지 않고 단계별로 관리합니다.
  • 3조건 도달과 실제 매도 완료는 다릅니다. 완료는 사용자가 직접 확인합니다.
  • 4이미 완료한 단계는 가격이 다시 내려가도 유지됩니다.
  • 5다음 단계 가격을 도중에 감정적으로 낮추지 않습니다.
  • 6남은 수량도 별도 규칙 없이 방치하지 않습니다.
// INPUT
계획 비율 합계: 0%
장중 도달은 실행이 빠르지만 순간 급등에도 작동합니다. 종가 확인은 일시적 움직임을 걸러내지만 장중 고가를 놓칠 수 있습니다. 어느 쪽이 맞는지는 본인의 계획에 달렸습니다.
입력값이 바뀌었습니다. 다시 판정해주세요.
선택의 장단점 (숫자가 아니라 구조)
단계가 적으면 단순하지만 한 시점의 판단 비중이 커집니다.
단계가 많으면 매도 시점 위험을 나눌 수 있지만 관리가 복잡해집니다.
매도 비율이 크면 수익을 더 많이 확정하지만 남은 상승 참여가 줄어듭니다.
매도 비율이 작으면 상승 참여는 유지되지만 수익 반납 가능성도 남습니다.
장중 도달 방식은 실행이 빠르지만 순간적인 가격 급등에도 작동할 수 있습니다.
종가 확인 방식은 일시적 움직임을 걸러내지만 장중 높은 가격을 놓칠 수 있습니다.
분할 매도 규칙 5가지
  1. 매도 가격과 비율은 매수 시점에 함께 정합니다.
  2. 전부 팔거나 끝까지 버티지 않고 단계별로 줄입니다.
  3. 조건 도달과 실제 체결은 다릅니다. 완료는 직접 확인합니다.
  4. 종가 기준이면 장중 가격으로 판정을 바꾸지 않습니다.
  5. 남은 수량도 방치하지 않고 별도 규칙으로 관리합니다.
매도는 최고점을 맞히는 기술이 아니라, 얻은 수익을 실제 계좌에 남기는 기술입니다.

이 도구는 사용자가 직접 설정한 분할 매도 계획에 숫자를 대입해 현재 단계를 보여주는 참고 도구입니다. 특정 종목의 매수, 매도 또는 수익을 보장하지 않으며, 가격이나 비율을 추천하지 않습니다. 조건 도달 표시는 실제 주문 체결을 의미하지 않으며, 최종 판단과 주문 실행의 책임은 사용자에게 있습니다.

작성 예시

가상의 연습 사례입니다. 권장 수익률이나 권장 매도 비율이 아닙니다. 이 숫자는 실제 입력창에 채워지지 않습니다.
